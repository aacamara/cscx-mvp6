## Codebase Patterns
- Use Tailwind CSS with CSCX custom colors: cscx-accent (#e63946), cscx-black, cscx-gray-900, cscx-gray-800, cscx-gray-700, cscx-gray-400, cscx-gray-300
- API calls use `const API_URL = import.meta.env.VITE_API_URL || ''` and `getAuthHeaders()` from useAuth hook
- Components use functional React with hooks (useState, useEffect, useCallback)
- Dark theme: bg-cscx-black, bg-cscx-gray-900, text-white, text-cscx-gray-400
- Buttons: bg-cscx-accent for primary, border-cscx-gray-700 for secondary
- Cards: bg-cscx-gray-900 border border-cscx-gray-800 rounded-lg
